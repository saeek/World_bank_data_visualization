# 🌍 World Bank Data Visualization

## 📌 Overview
This project presents a detailed data analysis and visualization of World Bank datasets, focusing on understanding global development patterns across countries. The analysis is compiled as a structured PDF report containing visual insights and interpretations.

---

## 🎯 Objective
- To explore global economic and social indicators  
- To perform data analysis using real-world datasets  
- To visualize trends and relationships between key variables  
- To interpret insights in a clear and structured format  

---

## 📊 Dataset
- Source: World Bank Data  
- The dataset includes indicators such as:
  - GDP  
  - Population  
  - Life Expectancy  
  - Other development metrics  

---

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Jupyter Notebook
